# CyberTools

This repository is a small, organized collection of **tools and resources** for **red teaming**, **Active Directory**, **reverse engineering**, and **exploit development**.
Each entry includes a short description and a direct link.

> The main list of tools and resources comes from a language model (LLM) and was adapted for simple navigation.

## Purpose

Provide a **quick-access hub** for essential material used in offensive security and technical analysis workflows.
